# ZEMA TECHNOLOGIES

Site vitrine de ZEMA TECHNOLOGIES.

## Pages

- `index.html` : accueil
- `offres.html` : offres
- `produits.html` : Testor et CVBoost
- `process.html` : méthode de travail
- `contact.html` : formulaire de contact
- `admin.html` : espace admin local

## Déploiement

Le site est statique et peut être hébergé sur Vercel, GitHub Pages ou tout hébergement web classique.

Configuration Vercel recommandée :

- Framework preset : `Other`
- Build command : vide
- Output directory : `.`

## Note importante

L’espace admin actuel fonctionne localement avec `localStorage`. Pour une version publique en ligne, il faut connecter Supabase afin que les messages et statistiques soient centralisés et lisibles depuis n’importe quel appareil.
